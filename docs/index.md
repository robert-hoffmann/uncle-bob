# Uncle Bob

![Uncle Bob](/brand/uncle-bob.png)

Uncle Bob is a portable skill system for AI coding agents. Install it into a
project or user profile, then use its skills to make agent work easier to
route, review, pause, resume, and govern.

The goal is not to make agents follow one rigid process. The goal is to give
them a shared operating model: choose the right skill, keep small work small,
promote larger work into durable artifacts, and apply specialist guidance only
when the task actually needs it.

## What The Skills Bring

- `ub-quality` gives every task a readability, structure, and tradeoff baseline.
- `ub-workflow` chooses between direct work, lightweight specs, and initiatives.
- `ub-governance` keeps evidence and gate behavior lean until risk justifies escalation.
- `ub-authoring` and `ub-customizations` help build better reusable agent guidance.
- Python, TypeScript, Vue, Nuxt, CSS, and Tailwind skills apply domain-specific rules.

## Who It Helps

Developers use Uncle Bob to make agent-assisted changes easier to understand,
validate, and continue after interruptions.

Managers use it to see how agent work is controlled: planning depth, review
checkpoints, evidence expectations, and escalation paths are explicit.
