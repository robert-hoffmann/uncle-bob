# Evidence Baseline 2026-03

This baseline is date-stamped to `2026-03-04` and optimized for lean default governance.

## 1) Default Evidence Controls

1. deterministic merge-gate artifacts for each blocking run
2. confidence/release evidence expansion when risk profile requires it
3. machine-readable validation record for blocking runs
4. decision-memory checks for high-risk paths
5. claim-confidence checks for policy assertions

## 2) Lean vs Advanced

Lean default:

1. focused artifact set for changed scope
2. minimal mandatory audit fields
3. no broad compliance matrix by default

Advanced add-ons (opt-in):

1. expanded non-functional evidence
2. deeper retention/audit requirements
3. broader standards mapping appendices
